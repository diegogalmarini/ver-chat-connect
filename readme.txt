A professional and lightweight messaging integration tool for agency clients.

== Description ==
Corporate plugin developed by **Diego Raul Galmarini** for **VER AI Labs**.

This plugin allows agency clients to integrate a direct and optimized contact button, featuring the official company logo and customized messages from the administration panel.

== Installation ==
1. Upload the `ver-chat-connect` folder to `/wp-content/plugins/`.
2. Activate the plugin.
3. Go to Settings > VER Chat Connect to configure your details.
